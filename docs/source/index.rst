Welcome to genderAEye's documentation!
========================================
Detect the gender of a person with just a picture of their eye

This project uses the `UBIpr dataset <http://iris.di.ubi.pt/ubipr.html>`
Citation: "Chandrashekhar Padole, Hugo Proença; Periocular Recognition: Analysis of Performance Degradation Factors, in Proceedings of the Fifth IAPR/IEEE International Conference on Biometrics – ICB 2012, New Delhi, India, March 30-April 1, 2012. "

.. note::

   This project is under active development.
